## Recording Modes: What to Capture

### Full Screen Recording

<a href="#">Movavi Screen Recorder Mac</a> full-screen mode captures the complete display area
of the selected monitor. On Macs connected to external displays, any monitor can be selected
as the recording source. Retina displays are captured at their full resolution, and the export
resolution settings control whether the full Retina resolution or a standard HD resolution
is written to the output file.

### Custom Region Capture

The custom region tool draws a capture boundary over any rectangular area of the screen.
Region capture is useful for recording a browser window without capturing the rest of the
desktop, recording a specific application panel, or capturing a web application at a precise
resolution.

The region boundaries are adjustable after drawing — drag the edges to resize or drag the
entire frame to reposition before starting the recording.

### Application Window Mode

Select any open application window as the exclusive capture source. The recording follows the
window if it is moved during capture, maintaining the application content in frame without
capturing surrounding desktop content. Window mode produces the cleanest recordings for
software demonstrations where only the demonstrated application should appear.

## Webcam Overlay Options

<a href="#">Movavi Screen Recorder Mac</a> webcam capture options provide flexibility for
presenter-on-screen recording:

**Position control**: The webcam overlay appears in any corner of the recording frame.
Position is selected before recording begins and maintained throughout the session.

**Size adjustment**: The webcam overlay size is scaled from a small corner presence to a
larger inset that occupies more of the recording frame, depending on how prominent the
presenter should appear relative to the screen content.

**Shape options**: The webcam feed is displayed as a rectangle or as a circular crop,
matching the visual style preference for the content type.

**Toggle during recording**: The webcam overlay can be turned on or off during a recording
session without stopping the capture, allowing the presenter to choose when to show their
face on screen.

## Audio Recording Architecture

Capturing both system audio and microphone requires routing system audio to the recorder —
a function that macOS does not expose natively because audio output is normally sent directly
to the output device. <a href="#">Movavi Screen Recorder Mac</a> installs a virtual audio driver
(Movavi Virtual Microphone) that intercepts system audio output and routes it to the recorder
alongside the physical microphone input.

The two audio streams are mixed to a single audio track in the recording output by default,
or optionally saved as separate tracks in formats that support multi-track audio. Separate
tracks allow independent level adjustment in a video editor if further production work is planned.

**Microphone monitoring**: The microphone level meter in the recording controls shows the
input level in real time, allowing volume adjustment before recording begins.

**Audio source selection**: Any available audio input device — built-in microphone, USB
microphone, USB audio interface — is selectable as the microphone source from a dropdown
in the recording settings.

## Annotation Tools During Recording

Real-time annotation in <a href="#">Movavi Screen Recorder Mac</a> adds visual emphasis to
recorded content without post-production:

**Pen drawing**: Freehand annotation draws attention to any element on screen. Drawing appears
in the recording at the moment the pen moves, with configurable color and stroke width.

**Shape overlays**: Arrows, rectangles, and ellipses are placed over screen content with a
click and drag, highlighting specific interface areas, buttons, and on-screen information.

**Text annotation**: Text boxes placed at any position on the screen appear in the recording
as on-screen labels, explaining content without narrating every element aloud.

**Keystroke display**: An optional keystroke indicator shows keyboard shortcuts as visual
elements in the recording, making keyboard-driven workflows visible to viewers who can not
infer them from watching the screen alone.

## Scheduled and Automated Recording

<a href="#">Movavi Screen Recorder for Mac</a> scheduling system enables recordings to start
automatically without manual initiation:

**One-time schedule**: Set a specific date and time for the recording to begin and a duration
for the session. The recording starts automatically at the configured time.

**Recurring schedule**: Configure recordings to repeat on selected days — useful for recording
a weekly webinar, a recurring broadcast, or any regularly scheduled content.

**Duration limit**: Optionally set the maximum recording length. The recording stops automatically
after the specified duration, preventing runaway recordings from filling available disk space.

**Post-recording actions**: Configure what happens when a recording ends — save and close,
export automatically to a specified format, or prompt for review.
